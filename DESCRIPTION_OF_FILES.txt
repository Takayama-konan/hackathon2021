<run file>
→ 実行する必要があるファイル

・run
→ 実行すると色々といいことが起きるかも


<Manager>
→ オブジェクトを実行するファイル，オブジェクト達を管理することが目的

・StateManager
→ 画面遷移をマネージメントする

・CommandManager
→ コマンドを押したり引いたりすることを感知する

・SaveAndLoadManager
→ セーブやロードするマネジメント


<Scene>
→ StateManagerで遷移されるシーン

･InitialScene
→ ゲームをスタートすると最初に表示される画面

・HelpScene
→ 操作方法画面

・MenuScene
→ メニュー画面

・EquipmentScene
→ 操作画面

・GachaScene
→ ガチャ画面

・PlayScene
→ 実際にプレイするダンジョンの画面

・StageSelectScene
→ ステージ選択画面

・GameOverScene
→ ゲームオーバーする画面

・BattleScene
→ バトル画面

・GoToEatScene
→ 予約画面へ行くよ


<Other>
・DrawDisplay
→ ディスプレイする方法とかを記したファイル

・OperateDB
→ データベースを操作する方法とかを記したファイル

・CreateMaze
→ 迷路作成ファイル
